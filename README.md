# Automotive ECU Diagnostic Simulator

## Overview

A modular Automotive ECU simulator developed in C to simulate vehicle sensors, ECU logic, dashboard display, diagnostic logging, and CAN communication without requiring physical hardware.

## Features

- Vehicle Dashboard Simulation
- Sensor Data Simulation
- ECU Decision Logic
- Fault Detection
- Diagnostic Logging
- CAN Message Simulation

## Technologies

- C
- Git
- GCC
- CMake
- GDB
